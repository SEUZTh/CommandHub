# CommandHub v1.4.0

CommandHub v1.4 将产品定位从“命令历史搜索”推进到“个人命令知识库”。

## 亮点

- 命令元数据：支持 `favorite / alias / note / workspace / category`
- 结构化搜索：支持 `Favorites`、`Category`、`Workspace` 过滤
- 双栏 Launcher：结果列表与详情区同时可见，支持内联编辑
- Workspace 管理：Settings 页面可创建、重命名、删除 workspace
- 数据兼容：启动时自动迁移 V1.3 SQLite 数据，不丢历史命令和上下文

## 交互变化

- 单击结果行现在只负责选中，不再立即复制
- `Enter`：复制当前命令
- `Cmd + Enter`：复制并关闭窗口
- `Cmd + Shift + F`：收藏 / 取消收藏
- `Cmd + E`：聚焦 alias 输入框
- `Cmd + F`：聚焦搜索框

## 数据与隐私

- 本地数据库继续保存在：
  - `~/Library/Application Support/CommandHub/commandhub.sqlite`
- V1.4 仍然只在本地存储命令、上下文和元数据
- 浏览器自动化权限仍仅用于读取当前 Chrome / Safari 标签页 URL 以改进排序

## 已知边界

- `note` 只作为弱匹配和辅助说明，不是强召回字段
- `category` 目前是规则推导字段，不支持手工编辑
- Settings 当前只提供 workspace 管理，不是完整偏好设置中心
