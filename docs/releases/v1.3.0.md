# CommandHub v1.3.0

CommandHub 是一个 macOS SwiftUI 应用：后台监听剪贴板里的命令文本，并用全局快捷键快速搜索、重新复制（不会直接执行命令）。

## 亮点

- 上下文感知搜索：从 Chrome / Safari 当前标签页解析 `url / domain / env`，对结果进行重排
- 搜索范围：`All / Current Env Only` 一键切换（`Current Env Only` 严格按 `domain > env` 过滤）
- 本地持久化：SQLite 存储命令与上下文统计（`commands` + `command_contexts`）
- 交互体验：全局快捷键 `Command + Shift + V`，键盘上下选择、回车复制、`Esc` 关闭
- 发布产物：自动构建并上传 `.dmg` 与 `.sha256`（GitHub Actions）

## 权限与数据

- 仅在需要读取浏览器 URL context 时，macOS 才会提示自动化权限；拒绝不影响命令采集与搜索
- 本地数据库默认位置：`~/Library/Application Support/CommandHub/commandhub.sqlite`
